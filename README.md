# MCTM

## Getting started

```bash
git clone https://github.com/MArpogaus/MCTM.git && cd MCTM # clone repo into mctm folder and change working directy
mamba create -n mctm pip python==3.9.* # create new mamba (conda) env called mctm
mamba activate mctm # activate newly created env
pip install -r requirements_tf.txt jupyter_lab # install required pkgs + jupyter lab server
jupyter lab # start jupyter lab server
```
